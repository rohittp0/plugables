# Plugables

Gradle plugins for Android/Kotlin projects.

| Plugin | Description | Docs |
|--------|-------------|------|
| [viewmodel-stub](viewmodel-stub/) | Generates an interface and preview-safe stub from ViewModels annotated with `@ViewModelStub` | https://rohittp.com/plugables/viewmodel-stub |
| [typed-events](typed-events/) | Generates type-safe Kotlin analytics event functions from a YAML schema | https://rohittp.com/plugables/typed-events |
| [codeview](codeview/) | Generates an interactive HTML report of `@Preview` screens with per-composable click-to-IDE overlays | [codeview/README.md](codeview/README.md) |
